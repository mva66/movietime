My Flix


This application provides users with access to information about various movies, directors and genres. Users can also sign up, update their their personal information and create list of their favorite movies.



Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing a title, description, genre property, director property and image. Example:

{ "Genre": { "Name": "Drama", "Description": "Mode of fictional representation through dialogue and performance." }, "Director": { "Name": "James Cameron", "Bio": "A Canadian filmmaker, artist, and environmentalist, who is best known for making science fiction and epic films.", "Birth": "1954-08-16", "Death": " " }, "_id": "5e23cb4450ee527f72df67de", "Title": "Titanic", "Description": "The film is based on accounts of the sinking of the RMS Titanic", "ImagePath": "titanic.jpg", "Featured": true }

Get data about a movie genre /movies/genre/[Name] GET None A JSON object holding data about a movie genre. Example:

{ "Name": "Drama", "Description": "Mode of fictional representation through dialogue and performance." }

Get data about a director /movies/director/[Name] GET None A JSON object holding data about a director. Example:

{ "Genre": { "Name": "Drama", "Description": "Mode of fictional representation through dialogue and performance." }, "Director": { "Name": "David Frankel", "Bio": "an American director, screenwriter and executive producer.", "Birth": "1959-04-02", "Death": " " }, "_id": "5e23caef50ee527f72df67da", "Title": "The Devil Wears Prada", "Description": "A 2006 American comedy-drama film directed by David Frankel and produced by Wendy Finerman.", "ImagePath": "devil-wear-prada.jpg", "Featured": true }

Add new user /users POST A JSON object holding data about the user to add, structured like:

{ ID : Integer, Username : String, Password : String, Email : String, Birthday : Date }

A JSON object holding data about the user that was added, including an ID. Example:

{ "FavoriteMovies": [], "_id": "5e251d5d7397190a4ce3a931", "Username": "vilad", "Password": "v1lahi894", "Email": "vilad@gmail.com", "Birthday": "1989-10-15T00:00:00.000Z", "__v": 0 }

Update a user's information /users/[Username] PUT A JSON object holding data about the user to update, structured like:

{ Username: String, (required) Password: String, (required) Email: String, (required) Birthday: Date }

A JSON object holding data about the user that was updated, including ID and Favorites: Example:

{ "FavoriteMovies": [], "_id": "5e251d5d7397190a4ce3a931", "Username": "vilad", "Password": "v1lahi894", "Email": "vilad12@gmail.com", "Birthday": "1989-10-15T00:00:00.000Z", "__v": 0 }

Add movies to user's list of favorites /users/[Username]/movies/[MovieID] POST None A JSON object holding data about the movie added to user's favorites. Example:

{ "FavoriteMovies": [ "5e23cb0250ee527f72df67db" ], "_id": "5e251d5d7397190a4ce3a931", "Username": "vilad", "Password": "v1lahi894", "Email": "vilad12@gmail.com", "Birthday": "1989-10-15T00:00:00.000Z", "__v": 0 }

Remove movies from user's list of favorites /users/[Username]/movies/[MovieID] DELETE None A JSON object holding data about the updated user's favorites list. Example:

{ "FavoriteMovies": [], "_id": "5e251d5d7397190a4ce3a931", "Username": "vilad", "Password": "v1lahi894", "Email": "vilad12@gmail.com", "Birthday": "1989-10-15T00:00:00.000Z", "__v": 0 }

Remove user accounts /users/[Username] DELETE None A message indicating the user's account was deleted
